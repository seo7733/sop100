SECTION 01
검토 배경 및 목적
보고서 작성 배경과 분석 목적
부산소방학교 지휘역량강화센터(ICTC)에서 개발 중인 SOP100 웹 기반 지휘교육 플랫폼에는 AI 지휘참모 기능이 탑재되어 있다. 동 기능은 훈련생이 지휘통제절차(SOP 100~113) 관련 질의를 하면 AI가 실시간으로 전술적 조언을 제공하는 역할을 수행한다.
본 보고서는 차기 부서 보고에서 제기될 수 있는 "왜 Claude API를 사용하는가?"라는 질의에 대응하기 위해, 주요 3개 AI 모델을 소방 지휘교육 특성에 맞는 기준으로 비교 분석하고 선정 근거를 객관적으로 제시한다.

SECTION 05
결론 및 선정 근거
Claude(Anthropic) 선정에 대한 종합적 판단
▶ 선정 결론

Claude(Anthropic Sonnet 4.6)를 AI 지휘참모 모델로 선정한다. 소방 지휘교육 플랫폼의 핵심 요건인 복잡한 SOP 절차의 정확한 맥락 파악, 긴 문서 동시 처리, 세부 지시사항 완전 이행 분야에서 경쟁 모델 대비 독보적 우위를 보인다. Gemini는 비용이 최저가이나 한국어 소방 전문용어 처리 정확도와 Google 종속성 우려로 제외하며, ChatGPT는 전반적으로 양호하나 긴 문서 처리와 지시 준수 측면에서 Claude에 미치지 못한다.

  • Constitutional AI 기반 설계로 허위 지휘 정보 생성 위험 최소화
  • SOC 2 Type II 인증 완료 — API 사용 시 입력 데이터 학습 미사용 보장
  • 현재 플랫폼(Supabase pgvector) RAG 시스템과 연동 호환성 최고
  • 기관 계정 전환 시 API 키 교체만으로 즉시 대응 가능 (전환 비용 최소)
  • Sonnet 4.6 기준 단가 ChatGPT와 동등 ($3 / $15 per 1M tokens)

▶ 과금 방식 및 운영 관리 상세
💳
과금 방식 — 선불 크레딧제
API는 선불 크레딧 구매 방식. 후불 자동 청구가 아닌 크레딧 잔액에서 사용량만큼 차감됨. 크레딧이 소진되면 API 호출이 즉시 차단(429 에러)되며 서비스 중단 없이 추가 구매로 재개 가능.
🔄
자동 충전(Auto-reload) 설정 가능
Claude Console → Billing 메뉴에서 자동 충전 임계값과 충전 금액 설정 가능. 예: "잔액 $50 이하 시 $200 자동 충전"으로 설정하면 크레딧 소진 없이 서비스 연속 운영 가능. 단, 별도 설정 없으면 자동 충전 안 됨.
📅
한 달 후 자동 리셋 여부
월간 자동 리셋 없음. 크레딧은 구매일로부터 1년간 유효하며 월별 초기화 없이 누적 사용. 단, 월별 지출 한도(Monthly Spend Limit)는 Usage Tier에 따라 설정되며 매월 초 리셋됨. 미사용 크레딧은 기간 내 그대로 유지.
⚠️
사용량 한도(Usage Tier) 구조
신규 계정은 Tier 1(월 $100 한도)에서 시작. 누적 크레딧 구매액에 따라 Tier 2→4로 자동 상향. 월 $252 사용 계획이라면 Tier 1에서는 한도 초과로 서비스 중단 가능 → Tier 2 이상 사전 확보 필요 (크레딧 $40 이상 구매 시 자동 상향).
📊
사용량 모니터링
Claude Console → Usage 페이지에서 모델별 · 날짜별 토큰 사용량 및 비용 실시간 확인 가능. 예산 임박 시 이메일 알림 설정 지원. 관리자(Admin) API 키로 사용량 자동 집계 API도 제공.
💡
비용 절감 전략
Prompt Caching: SOP 시스템 프롬프트 반복 전송 없이 캐시 재사용 → 최대 90% 절감
Batch API: 실시간 응답 불필요 시 50% 할인
월 지출 한도 설정: Console에서 소비 상한선 직접 설정 가능
▶ Anthropic API 사용량 Tier 구조
Tier Tier 진입 조건 월 지출 한도 비고
